# Portfolio Website

A modern React + Vite portfolio showcasing my experience, projects, and expertise.

## Project Roadmap

### Phase 1: MVP (Core Sections)
- [ ] **Intro/Hero** - Introduction and value proposition
- [ ] **Navbar** - Navigation with smooth scrolling
- [ ] **Skills** - Technical skills and tech stack
- [ ] **TA Experience** - Teaching and mentoring background
- [ ] **Engineering Design Team** - Team leadership experience
- [ ] **Projects** - Showcase 3-5 best projects with links
- [ ] **Resume** - Downloadable PDF + one-page summary
- [ ] **Contact** - Email, LinkedIn, GitHub links

### Phase 2: Polish
- [ ] Dark mode toggle
- [ ] Smooth scroll animations (Framer Motion)
- [ ] Testimonials section (2-3 quotes)
- [ ] Mobile responsiveness refinement
- [ ] Performance optimization

### Phase 3: Stretch Goals
- [ ] Blog section with markdown support
- [ ] Project filtering by category/tech
- [ ] Newsletter signup
- [ ] Analytics integration
